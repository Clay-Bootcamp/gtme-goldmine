# Claude Code's Capabilities

[https://supercut.ai/share/claude-code-gtm/nVY2A8WRvaj3C6wkXXKozn](https://supercut.ai/share/claude-code-gtm/nVY2A8WRvaj3C6wkXXKozn)

Other AI tools (ChatGPT, Perplexity, even Claude on the browser) can tell you what to do. Claude Code actually does it for you. It writes the code, runs it, and gives you the end result.

### **What Claude Code can do**

- **Read and write your files** - edit CSVs, create documents, move things around on your computer
- **Run terminal commands** - even ones you've never heard of, on your behalf
- **Fix its own mistakes** - if something breaks, it figures out what went wrong and corrects itself
- **Connect to external services** - send data to Salesforce, HubSpot, Clay, or pull from Apollo, Apify, and more via APIs and MCPs

In the video, you'll see a live demo of all four of these in action using a simple prospects.csv file.

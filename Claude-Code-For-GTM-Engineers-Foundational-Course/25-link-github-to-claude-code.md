# How to link GitHub to Claude Code (A primer on Git for non-technicals)

> ## 📺 Video Lesson
> **▶️ [Watch the Supercut walkthrough](https://supercut.ai/share/claude-code-gtm/_nhfbL7N-NxVYnfuz9_xQA)**
>
> This lesson has a video. Watch it first, then use the prompt below.

The prompt to paste into Claude Code:

```
Help me connect this terminal to GitHub so I can push files. I'm not
technical — go one step at a time, wait for my confirmation after each
step, and explain anything confusing in plain English. If you need me to
log in to GitHub via the browser, give me the exact command to run in a
separate terminal window and walk me through the auth flow. If I don't
have a GitHub account yet, help me make one first. When done, verify
everything is connected.
```

# Claude Code For GTM Engineers — Foundational Course

**Go from zero to shipping real GTM systems with Claude Code — no coding background required.**

This is a self-guided course built for go-to-market engineers, operators, and founders who want to build software by *talking*, not by learning to code. Work through it top to bottom: each module builds on the one before it. By the end you'll have Claude Code installed, a repeatable prompting framework, and a working mental model for memory, skills, agents, integrations, security, and Git.

> 🎥 **Many lessons have a video walkthrough (a "Supercut").** When you see the **📺 Video Lesson** box at the top of a lesson, watch it first — then use the written notes as your reference and cheat sheet.

---

## How this course works

- **It's linear.** Start at Lesson 1 and go in order. Later lessons assume earlier ones.
- **Every lesson has navigation.** Use the **← Previous / 🏠 Course Home / Next →** bar at the top and bottom of each lesson to jump in and out.
- **🎥 marks a video lesson.** Watch the Supercut, then read.
- **Reference modules (5–9) are also lookups.** Once you've been through them once, come back and read just the one you need when you hit the problem it solves.
- **Total time:** ~2–3 hours to go through everything once. Setup alone is ~30 minutes.

**Legend:** 🎥 has a video walkthrough · ⏱ estimated time · 📌 reference / look-it-up lesson

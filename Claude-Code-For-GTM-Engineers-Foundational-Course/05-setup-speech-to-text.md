# Set up Speech-to-Text

## Why You Need This

You'll be *talking* to Claude Code, not typing. Speaking is 3-4x faster than typing and feels more like a natural conversation. This is how we'll work during the course.

You need a way to turn your voice into text in any app.

Have one of these ready:

**Mac:**

- [Wispr Flow](https://wisprflow.ai/) (paid, excellent)
- [Monologue](https://www.monologue.to/) (free tier available)
- Built-in: System Settings → Keyboard → Dictation → On

**Windows:**

- Built-in: Press Win + H to activate voice typing

**✓ Success looks like:** You can speak and see your words typed on screen.

## Installation

### Mac Options

- **Option 1: Wispr Flow** (paid and excellent, recommended)
    1. Go to [https://wisprflow.ai](https://wisprflow.ai/)
    2. Download and install
    3. Follow their setup guide
- **Option 2: Monologue (g**ood free option)
    1. Go to [https://monologue.to](https://monologue.to/)
    2. Download and install
    3. Create an account (free tier available)
- **Option 3: Built-in Mac Dictation**

    Free, already on your Mac. Less accurate but works.

    1. Open **System Settings**
    2. Go to **Keyboard**
    3. Scroll down to **Dictation** → Turn it **On**
    4. To use: Press the microphone key (Fn twice by default) in any app

### Windows Options

- **Built-in Voice Typing**
Windows has this built in. Works well enough.
    - Press **Win + H** to activate voice typing in any app.
    - That's it. No installation needed.

## ✓ Verify It Worked

1. Open any text app (Notes, Word, or just a browser search box)
2. Activate your voice input
3. Say "Testing one two three"

**Success looks like:** Your words appear on screen as you speak.

## Troubleshooting

**Mac Dictation: "Dictation couldn't start"**

- Check System Settings → Privacy & Security → Microphone → Make sure your apps have
permission

**Wispr/Monologue not hearing you**

- Check that you've granted microphone permissions when prompted
- Check System Settings → Privacy & Security → Microphone

**Windows: Win+H not working**

- Make sure you're on Windows 10 or 11
- Go to Settings → Time & Language → Speech → Turn on voice typing

# Commit & Restore

> ## 📺 Video Lesson
> **▶️ [Watch the Supercut walkthrough](https://supercut.ai/share/claude-code-gtm/7DHZR5x7m8GnrxvFpP2eBF)**
>
> This lesson has a video. Watch it, then do the exercise below.

> ✏️ **Exercise:** Commit a project to GitHub. Follow along with the video, break things by locally editing the files in a text editor or code editor (Notepad or TextEdit, for example), make sure the page looks broken when you double-click the HTML file. Then ask Claude to restore it.
>
> Make a tiny change to your page, and then ask Claude Code to commit it to GitHub.
>
> Once you finish this exercise, you can rest easy knowing your work is always safe and backed up.

> 🟥 To **"break the page"** use any software on your computer that has the ability to edit HTML files — or any code editor — for example Notepad, TextEdit, Cursor, or anything else.
